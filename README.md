# Kube Node Pool

通过 Nuxt 实现的 kubernetes 节点池控制台，功能如下:

1. **向导**：快速在腾讯云上通过抢占式服务器创建 kubernetes 集群。
   1. 也可以不是抢占式服务器，主要取决于你的服务器模板。
   2. 也可以不是腾讯云，可以添加其他服务商的接口。
2. **后台服务**：检查服务器数量的，在云服务器被释放时创建新的节点加入集群。
3. **仪表盘**：查看应用和节点池状态


更具体的功能说明见[需求文档](./docs/requirements.md)
